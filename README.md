# Her House 팀

## 💻 프로젝트 내용

- React를 사용한 오늘의 집 클론 코딩
- CRA를 사용한 초기 세팅
- 함수형 컴포넌트, styled components 사용
- 상품화면, 시공업체 페이지를 메인 아이템으로 채택

## 💻 프로젝트 기간

2021.04.26 ~ 2021.05.07
